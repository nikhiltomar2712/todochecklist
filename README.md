## Features

- Add tasks with the `Enter` key or the `+` button
- Mark tasks complete with a custom checkbox and strikethrough style
- Delete individual tasks with a smooth exit animation
- Filter tasks by **All**, **Active**, or **Completed**
- Clear all completed tasks in one click
- Timestamps showing when each task was added
- All tasks persist in `localStorage` across page refreshes
- Fully responsive — works on mobile and desktop

## Getting Started

No install or build step needed. Just open `index.html` in your browser.

```bash
git clone https://github.com/your-username/taskflow.git
cd taskflow
open index.html
```

Or use a local server for development:

```bash
# Python
python -m http.server 5500

# Node
npx serve .
```
